## 🚀 About

OJOL is a revolutionary AI-powered ride-hailing platform designed to create a fairer, more efficient ecosystem for both drivers and users. By leveraging artificial intelligence and blockchain technology, we've reduced the operational fee to just **7%**, allowing drivers to keep 93% of their earnings.

### Key Features

- **🤖 AI-Powered Dispatch System** - Smart matching algorithms for optimal ride allocation
- **💰 Low 7% Operational Fee** - Industry-leading fee structure for driver earnings
- **🌍 Multi-Language Support** - English, Indonesian, and Chinese
- **📱 Dual-Platform Ecosystem** - Separate apps for riders and drivers
- **💎 OJOL Coin** - Utility token for rewards, incentives, and governance
- **🎯 4M+ Drivers & 88M+ Users** - Massive Indonesian market opportunity

## 🛠️ Installation & Setup

### Prerequisites

- Node.js 18+ and npm/yarn/bun
- Git

### Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/ojol-website.git
cd ojol-website
```

### Install Dependencies

```bash
# Using npm
npm install

# Using yarn
yarn install

# Using bun
bun install
```

### Run Development Server

```bash
# Using npm
npm run dev

# Using yarn
yarn dev

# Using bun
bun run dev
```

The application will be available at [http://localhost:3000](http://localhost:3000)

### Build for Production

```bash
# Using npm
npm run build

# Using yarn
yarn build

# Using bun
bun run build
```

### Start Production Server

```bash
# Using npm
npm run start

# Using yarn
yarn start

# Using bun
bun run start
```
